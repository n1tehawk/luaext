luaext

Lua extensions library

luaext is a collection of Lua modules that add useful extensions to the standard set of Lua functions / libraries. See the GitHub project page.

To start using luaext, a simple require("luaext") should include the luaext module and make the functions available. (This is configuration dependent, see the module description for further details.)
