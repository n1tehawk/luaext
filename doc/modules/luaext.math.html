Module luaext.math

Mathematical functions.

This file is part of the luaext library.

If USE_LUA_NAMESPACES is configured, the module functions will also get exported to the standard math namespace.

Functions

frac (n) return the fractional part of a number (preserving the sign).
round (n, decimals) rounds a number.
trunc (n) return the integral part of a number (rounded towards zero).


Functions

frac (n)
return the fractional part of a number (preserving the sign). See also: trunc, math.modf()

Parameters:

  • n number

Usage:

    assert(luaext.math.frac(1.98) == 0.98)
assert(luaext.math.trunc(-1.75) == -0.75)
assert(luaext.math.frac(1234) == 0)
round (n, decimals)
rounds a number. This function will perform rounding of a value to the given number of decimal places (precision), using a “round half up” algorithm (see Wikipedia). Note: It’s even possible to use negative values of decimals to round to higher powers of ten.

Parameters:

  • n number the value to round
  • decimals optional number precision (number of decimal places) to keep after rounding. defaults to 0, which rounds to an integer value.

Usage:

    assert(luaext.math.round(math.pi, 2) == 3.14)
assert(luaext.math.round(500, -3) == 1000) -- round to 1E3 = thousands
trunc (n)
return the integral part of a number (rounded towards zero). See also: frac, math.modf()

Parameters:

  • n number

Usage:

    assert(luaext.math.trunc(math.pi) == 3)
assert(luaext.math.trunc(-1.75) == -1)
